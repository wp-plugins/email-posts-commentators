Plugin to email commentators of posts

== Description ==

Plugin offers a admin screen to select the posts of which commentators should be emailed, and then you can write the subject & message to send and the plugin will send them an email along with their comment history on the selected posts.

You also have the option of excluding list of emails and specifying a Bcc email address.

== Installation ==

1. Upload `email-posts-commentators` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to `Tools` > `Email Posts Commentators` to send emails.
